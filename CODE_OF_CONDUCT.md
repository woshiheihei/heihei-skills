# Code of Conduct

## Our standards

We want this project to feel useful, respectful, and safe for contributors and
users. Examples of behavior that support that environment include:

- being respectful in language and tone
- giving actionable, evidence-based feedback
- welcoming different levels of experience
- assuming good intent while still addressing concrete problems
- focusing discussion on the repository, workflow, and user outcomes

Examples of unacceptable behavior include:

- harassment, discrimination, or personal attacks
- doxxing or sharing private information without permission
- trolling, intimidation, or sustained disruption
- bad-faith review behavior or hostile gatekeeping
- publishing sensitive security details in public channels

## Scope

This code of conduct applies to issues, pull requests, discussions, and other
project spaces connected to this repository.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull
requests, or other contributions that violate this code of conduct. Repeated or
severe violations may lead to temporary or permanent exclusion from project
spaces.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer
through a private contact channel listed on the repository owner's GitHub
profile. Please avoid posting personal or sensitive details in public threads.
