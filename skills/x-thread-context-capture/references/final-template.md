# Deliverable template

Use this as a structural template, not as boilerplate to repeat mechanically. Keep the headings stable unless the task clearly requires a variant.

## Suggested structure

Default filename:

`x-thread-context-<author_handle>-<root_status_id>.md`

Default output path:

`./x-thread-contexts/x-thread-context-<author_handle>-<root_status_id>.md`

## Notes

- Keep the document single-file and self-contained.
- Preserve raw text over paraphrase.
- Use one subsection per important author reply or important comment.
- If no author follow-ups are visible, keep the section and state that none were found after searching.
- Default to the structured output path above unless the user explicitly asks for a different location or filename.
